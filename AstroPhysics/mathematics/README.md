# Mathematics

Interactive math-oriented tools and supporting documents.

## Contents

- `docs/`: moved math/physics document folder
- `newton_function_animation.py`: enter a function and animate Newton's method solving for a root

## Run

```bash
python3 AstroPhysics/mathematics/newton_function_animation.py
```

Example expressions:

- `x**2 - 4`
- `x**3 - x - 2`
- `np.cos(x) - x`
- `np.sin(x) - 0.25`

Controls:

- `q` or `Esc`: quit
- `p` or `Space`: pause
- `r`: restart animation from the first Newton step
- `n`: enter a new function
