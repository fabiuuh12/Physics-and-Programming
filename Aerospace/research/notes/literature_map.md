# Literature Map

## Cislunar autonomous navigation

- NASA and Aerospace Corporation work on cislunar autonomous navigation and
  onboard navigation architectures.
- Research on autonomous orbit determination in CR3BP and cislunar regimes.
- Cooperative navigation using inter-satellite range and angle measurements.

## Filtering methods

- Extended Kalman Filter (EKF).
- Unscented Kalman Filter (UKF).
- Cubature Kalman Filter (CKF).
- Adaptive and robust filters for measurement outages and non-Gaussian noise.

## Optical/ranging measurements

- Optical line-of-sight navigation.
- Range and range-rate measurements.
- Geometry strength and observability metrics.
- Lighting, eclipse, glare, and target visibility.

## Research gap for this project

This project tests whether a simple interpretable navigation risk score can
improve EKF robustness under cislunar stress cases while staying transparent
enough for a laptop-scale research simulation.
